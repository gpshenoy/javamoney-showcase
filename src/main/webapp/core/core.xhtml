Core
API for handling and calculating with monetary amounts.Support different numeric capabilities and guarantees to be provided by the monetary amount implementations. These data is called monetary context and must be accessible from an amount instance during runtime. Extension points for adding additional logic, e.g. for extending the arithmetic capabilities, rounding etc. API to externalize the numeric part of an amount to the most useful representation on a runtime platform. API to to create a new amount instance using an existing amount as a template, hereby changing currency and/or numeric part as required. This ensures maximal portability and allows externalization of complex financial calculations. Provides a minimal set of roundings and also supports arbitrary custom roundings.
